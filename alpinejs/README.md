# Alpine.js Buttondown Example

A simple newsletter subscription form built with Alpine.js and Buttondown.

## Getting Started

1. Clone this repository
2. Replace `{username}` in `index.html` with your Buttondown username
3. Run the development server:

```bash
npm run dev
```

4. Open http://localhost:8000 in your browser

## Features

- Simple email subscription form
- Loading states during submission
- Client-side form handling with Alpine.js
- No build process required

## Notes

This example uses Alpine.js from CDN, so no npm installation is required. The HTML file can be opened directly in a browser or served via any static file server. It's iframe-ready and requires no build process.
