# HTML Buttondown Examples

Newsletter subscription form examples built with plain HTML.

## Getting Started

1. Clone this repository
2. Replace `{username}` in the HTML files with your Buttondown username
3. Open the HTML files directly in your browser

## Examples

- `basic.html` - Simple email subscription form with email only 
- `dropdown.html` - Subscription form with language selection dropdown (submits as `tag`) 
- `checkbox.html` - Subscription form with topic checkboxes (multiple tags)
- `hidden.html` - Subscription form for pre-tag subscribers with a hidden field 
- `name.html` - Minimalist form to collect names 

## Features

- No build process required
- iframe-ready
- Form validation
- Subscription form with language selection dropdown
